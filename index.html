PCL (Python-C Linked) is a minimal experimental compiler that lets you write both Python and C code in a single .pcl file and execute them together.

This online playground is powered by a Flask backend hosted on Render (Free Tier), allowing real-time execution and testing directly in your browser.

Created by Muhammed Shafin P. Contribute on GitHub!

%c name=mathmod export=add,g_counter
#include <stdio.h>

int g_counter = 0;

int add(int a, int b) {
    return a + b;
}
%endc

%py requires=mathmod
print("Sum 3 + 4 =", add(3, 4))
print("Counter before increment:", g_counter.value)
g_counter.value += 1
print("Counter after increment:", g_counter.value)
%endpy
